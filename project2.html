Description

Pour ce projet, j'ai dû travailler en équipe avec Loïc Tuplink et Antoine Sprumont. Nous étions dans une industrie Lego et nous devions trier une sorte de brique selon des critères. Notre groupe a dû trier les briques en fonction de leur forme.

Pour ce projet, nous avons dû utiliser des systèmes embarqués, raspberry pi et des capteurs. J'ai aussi fait un modèle 3D pour le bouton afin de les mettre dans le rail Din. Les briques Lego ont été placées sur un tapis roulant pour les acheminer vers notre système. Les capteurs utilisés étaient une caméra à intelligence artificielle et un capteur infrarouge. La caméra a été utilisée pour reconnaître la forme de la brique et le capteur infrarouge a été utilisé pour détecter quand nous devions envoyer l'information au Raspberry Pi.

Le raspberry pi a été utilisé pour envoyer les informations à la base de données et à la page web, pour ce faire nous avons un docker avec un nodered qui nous a aidé à envoyer les informations à la base de données. Après cela, le site Web récupère les données de la base de données pour les afficher sur la page Web. Le site web était codé avec nodejs et la base de données était un mysql.

Nous avons eu deux problèmes récurrents : Le premier était la caméra : elle ne fonctionnait pas bien, à certains moment parce qu'il détectait la mauvaise forme de la brique. Le second était le tapis roulant avec notre système intégré. La bande transporteuse interférait avec le système embarqué, nous avons donc dû placer la masse du système embarqué sur le cadre métallique de la bande transporteuse.

À la fin du projet, nous devions faire une présentation de notre projet et nous devions montrer comment il fonctionnait. Cela fonctionnait bien et tous les objectifs étaient atteints. Grâce au projet, nous avons appris à combiner toutes nos connaissances pour mener à bien ce projet.
